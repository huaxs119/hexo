2. 比较运算符

== 比较对象是否相等
!= 比较对象是否不相等
<> 是否不相等

< 是否小于
>= 大于等于
<= 小于等于

3. 逻辑运算

and 与 x and y x为false则返回x，否则返回y
or 或 x or y x为false返回y，否则返回x
not 非

4. 运算符优先级
