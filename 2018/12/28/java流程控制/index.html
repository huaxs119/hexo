java流程控制

发表于 2018-12-28 | 分类于 开发

条件判断

if

if(判断条件){
          执行的代码块1;
          执行的代码块2;
}

if…else

if(判断条件){
          执行代码块1
}else{
          执行代码块2
}

if…else if…else

if(判断条件1){
          执行代码块1
}else if(判断条件2){
          执行代码块2
}else{
          执行代码块3
}

选择判断

switch…case…default

switch(表达式){
          case 取值1:
                  执行语句;
                  break;
          case 取值2:
                  执行语句;
                  break;
          default:
                  执行语句;
                  break;
}

switch语句的特点：

  1. 选择的类型只有四种：byte，short，int，char
  2. 先判断所有的case，没有匹配的case就执行default
  3. case后跟的值必须是常量，不能是变量

循环

while

while(条件表达式){
           执行语句;
}

do…while

do{
    执行语句;
}while(条件表达式)
无论条件是否满足，循环体至少被执行一次

Math.random() 生成0-1之间的数字，double型
Math.random()10 生成0-10之间的数字，double型
(int)(Math.random()10) 强转int类型，去掉小数点，取0-9，便于计算
(int)(Math.random()*10)+1 生成1-10之间的随机数

for

for(初始表达式;循环表达式;操作表达式){
    执行语句;
}

导致无限循环的格式while(true)和for（;;）

break和continue

break

终止最近的封闭循环，不会执行后面的语句，可以单独使用，也可以与标签一起使用

continue

跳出本次循环，执行下一次
